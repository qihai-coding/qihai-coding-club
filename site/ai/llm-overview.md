# LLM 概览

## 核心概念

- Token：模型处理的最小文本单元。
- 上下文窗口：单次请求可使用的最大 Token 数。
- Embedding：用于语义检索的向量表示。
- Temperature：控制生成结果随机性的参数。

## 常见 RAG 流程

1. 将文档切分为多个片段。
2. 对片段向量化并写入向量库。
3. 按查询语义相似度召回相关片段。
4. 将召回内容拼接进提示词。
5. 生成答案并附带引用信息。
