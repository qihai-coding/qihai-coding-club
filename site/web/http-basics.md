# HTTP 基础

## 请求

- 方法：`GET`、`POST`、`PUT`、`DELETE` 等。
- URL：协议、主机、路径、查询参数。
- 请求头：例如 `Content-Type`、`Authorization`。
- 请求体：携带业务数据的内容。

## 响应

- 状态码表示结果（如 `200`、`404`、`500`）。
- 响应头描述缓存策略与内容类型等行为。
- 响应体可以是 JSON、HTML、文件或纯文本。
